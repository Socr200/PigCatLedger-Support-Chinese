欢迎使用 **猪咪记账**！本页面为您的问题提供快速解答与帮助。

## 一、应用简介

猪咪记账是一款专注于日常收支管理的 iOS 应用，主要功能包括：
- **AI 智能记账**：拍照或输入文字，自动识别交易类型、金额、分类。  
- **多账本管理**：支持创建和切换多个账本，满足不同场景需求。  
- **专业版解锁**：购买专业版后可享受无限账本、高级账户管理、CSV 导入/导出、iCloud 多端同步等功能。  
- **打赏支持**：可通过小额打赏鼓励我们持续改进。

---

## 二、安装与升级

1. **下载**  
   直接从 App Store 搜索 “猪咪记账” 点击下载。  
2. **升级到专业版**  
   - 在「设置 → 升级专业版」页面点击购买按钮。  
   - 完成 App Store 内购买后即可立即解锁所有高级功能。  
3. **恢复购买**  
   如果更换设备或重装应用，请在「设置 → 恢复购买」中点击即可恢复已购专业版。

## 三、常见问题（FAQ）

### 1. 为什么 AI 自动记账识别不准确？  
- 请确保文字/照片清晰可见；  
- 尝试手动修改识别结果后再保存，系统会学习改进。

### 2. 如何重置打赏购买？  
打赏为消耗型内购，不支持恢复；若操作失败可直接再次购买，无重复扣费提示。

### 3. 专业版激活后没有生效怎么办？  
- 检查您的网络连接；  
- 在「设置 → 恢复购买」中尝试恢复；  
- 如仍未生效，请联系我们。

### 4. 如何备份和导出数据？  
- 在「设置 → 导入导出」中选择 CSV 导出；  
- 通过邮件或其他方式保存导出文件。

## 四、使用指南

1. **创建账本**  
   - 点击「账本列表」右上角“＋”按钮，新建或选择已有账本。  
2. **添加交易**  
   - 在「交易列表」点击“＋”按钮；  
   - 选择「AI 记账」或「手动输入」；  
3. **查看报表**  
   - 在底部导航切换到「报表」页，查看收支概况、分类饼图等。  
4. **打赏支持**  
   - 在「设置 → 支持或打赏 ❤️」页面，选择打赏金额，完成内购即可。
